## 🚀 La Evolución de la Música Local

Olvídate de los reproductores estándar, aburridos y genéricos. **Listenfy** ha sido diseñado meticulosamente en la convergencia entre arte visual y potencia tecnológica. No es solo un lugar para escuchar tus archivos; es un **centro de entretenimiento** que transforma tu biblioteca local en una experiencia viva. 

Diseñado para las pantallas de hoy con tipografías vibrantes, cristales esmerilados, animaciones ultra-fluidas y un **motor de IA integrado** que moldea el sonido a tu gusto. Si exiges lo mejor para tu dispositivo, **Listenfy es para ti**.

## 🛠️ Arquitectura High-End (Para Desarrolladores)

Si vas por debajo del capó, Listenfy es una obra maestra técnica diseñada bajo la arquitectura Feature-First / Modular Pattern, garantizando un escalamiento infinito sin deuda técnica.

- **Frontend Core:** `Flutter`, Dart 3.10+, adaptativo multi-pantalla, Material You dinámico.
- **Media Engine:** `just_audio` en total sintonía con `audio_service` (Lockscreen/Background intacto), `video_player`.
- **Ecosistema GetX:** Reactividad de nanosegundos (Rutas, Obx, Dependencias estables).
- **Almacenamiento I/O Cero Latencia:** `Hive` estructurado por Cajas de memoria combinadas con `GetStorage`.
- **Hardware y Nativas:** `permission_handler` avanzado y `nearby_connections` P2P.

## ⚡ Instalación

Construir la app desde el código fuente es sencillo si tienes instaladas las herramientas de Flutter.

1. **Clona de la bóveda principal:**
   ```bash
   git clone https://github.com/tu-usuario/listenfy.git
   cd listenfy/Front
   ```

2. **Instala las dependencias secretas:**
   ```bash
   flutter pub get
   ```

3. **Lanza y deslumbra:**
   (Asegúrate de conceder permisos de Almacenamiento, Bluetooth para audífonos y Notificaciones).
   ```bash
   flutter run
   ```
